# VELENEŚIA

A Modern Online Clothing Shopping & Management System

## Overview

VELENEŚIA is a modern web-based clothing shopping and management system developed using React.js. The platform provides customers with an interactive shopping experience while enabling administrators to efficiently manage products and customer orders.

The project demonstrates the implementation of modern web engineering concepts including component-based architecture, client-side routing, state management, responsive design, and dynamic user interactions.

## Features

### Customer Features

* User Registration and Login
* Browse Clothing Collections
* Product Search and Filtering
* Product Details View
* Shopping Cart Management
* Secure Checkout Flow
* Multiple Payment Options
* Order Tracking

### Admin Features

* Product Management
* Add, Update, and Delete Products
* Order Management
* Update Order Status
* Inventory Monitoring

## Technology Stack

### Frontend

* React.js
* React Router
* Context API
* Tailwind CSS
* CSS3
* JavaScript (ES6+)

### Development Tools

* Node.js
* npm
* VS Code

## System Architecture

The application follows a component-based architecture where reusable React components are used to maintain scalability, modularity, and maintainability.

Client-side routing is implemented using React Router, while application state is managed through Context API to ensure a consistent user experience across different pages.

## Project Objectives

* Improve customer convenience through online shopping
* Provide dynamic product browsing capabilities
* Simplify shopping cart and checkout processes
* Enable efficient product and order management
* Deliver a responsive and user-friendly interface

## Installation

1. Clone the repository

```bash
git clone https://github.com/mubashranoor04/velenesia-clothing-store.git
```

2. Navigate to the project folder

```bash
cd velenesia-clothing-store
```

3. Install dependencies

```bash
npm install
```

4. Run the application

```bash
npm start
```
